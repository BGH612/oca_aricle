# Solana Exchange-Flow Forecasting

This repository contains the code and prepared datasets used for the empirical analysis of Solana exchange-flow on-chain indicators in short-term SOL price-direction forecasting.

## Data

The `datasets/` folder contains the prepared modelling datasets used in the article:

- `merged_base.csv` - baseline market, macroeconomic, search-interest and VMD features.
- `merged_oca.csv` - Solana exchange-flow on-chain features.
- `merged_ta.csv` - technical analysis features.
- `baseline_oca.csv` - baseline and OCA features.
- `baseline_ta.csv` - baseline and TA features.
- `oca_ta.csv` and `merged_oca_ta.csv` - OCA and TA features.
- `full.csv` - full multimodal specification combining baseline, OCA and TA features.

The target variable is constructed as the next-period direction of SOL price movement.

## Main Experiment Pipeline

The main reproducible pipeline is:

```bash
python scripts/run_article_ready_hypothesis_pipeline.py
```

It runs the article-ready forecasting experiment, including:

- classification models for SOL direction forecasting;
- block-level ablation analysis;
- internal OCA ablation and feature-importance diagnostics;
- out-of-sample holdout evaluation;
- trading-strategy evaluation;
- bootstrap confidence intervals;
- McNemar and Diebold-Mariano tests;
- regime-dependent performance checks.

## Data Collection and Feature Engineering

The `data_collection/` folder contains notebooks used to prepare the modelling datasets:

- `solana_searching_transfers_clear.ipynb` - Solana transfer collection and cleaning.
- `prepairing_oca_dataset.ipynb` - construction of OCA exchange-flow features.
- `indicators_collection.ipynb` - technical, macroeconomic, search-interest and VMD feature preparation.
- `main_pipeline_graphs.ipynb` - pipeline implementation, visualizations and article figures.

## Feature Blocks

The empirical analysis compares the following feature specifications:

- `baseline` - market, macroeconomic, search-interest and VMD controls.
- `OCA` - Solana exchange-flow on-chain features.
- `TA` - technical indicators.
- `baseline + OCA`
- `baseline + TA`
- `OCA + TA`
- `full` - all feature blocks combined.

## Notes

The repository is intended to make the empirical part of the article reproducible. Some notebooks were used during exploratory analysis, while the main article results are generated by the scripts in `scripts/`.
